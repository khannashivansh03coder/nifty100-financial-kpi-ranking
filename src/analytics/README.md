# 📈 Nifty 100 Financial KPI Ranking & Analytics Dashboard

## Overview

The Nifty 100 Financial KPI Ranking project is an end-to-end financial analytics system developed to analyze publicly available financial statements of Nifty 100 companies.

The project integrates ETL pipelines, financial KPI computation, company ranking, valuation analysis, peer comparison, and an interactive Streamlit dashboard.

---

## Features

* ETL pipeline for financial datasets
* Data validation and preprocessing
* Financial KPI calculations
* Company ranking engine
* Peer comparison dashboard
* Sector analytics
* Trend analysis
* Capital allocation visualization
* Valuation engine
* Streamlit interactive dashboard
